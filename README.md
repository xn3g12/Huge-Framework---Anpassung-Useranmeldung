Demo- und Übungsanwendung
In dieser Einheit habe ich das Captcha entfernt und ein paar Benutzer mit Name und E-Mail angelegt. Ich konnte mich auch erfolgreich einloggen. Ich habe außerdem versucht, einen Benutzer ohne E-Mail anzulegen, was ebenfalls funktioniert hat. Die Benutzer konnten unter „Profile“ eingesehen werden, was allerdings etwas problematisch ist, da beim Einloggen eine Meldung erscheint, dass eine E-Mail erforderlich ist.
